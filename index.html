학생이 선생님을 찾기 위해 교무실을 직접 방문해야 하는 불편을 줄이기 위해, 교무실 자리 배치와 교사 상태를 실시간으로 공유하는 웹 시스템을 구현했다. 재실 상태인 교사에게만 호출을 보낼 수 있고, 교무실 화면에서는 호출을 크게 표시하고 음성으로 안내한다.

문제 출발점
선생님이 자리에 계신지 알 수 없어 학생이 교무실을 헛걸음하거나, 교무실 내부에서 특정 교사를 반복해서 찾는 문제
주요 기능
교무실 자리 배치 시각화 · 교사 상태 표시 · 재실 조건부 호출 · 호출 현황판 · TTS 음성 안내 · 관리자 편집
화면 구성
board.html 학생/현황판 · teacher.html 교사 상태 변경 · admin.html 자리/교무실 관리 · functions TTS 프록시
핵심 구현
Firebase Realtime Database 구독으로 상태·호출·자리 변경을 새로고침 없이 모든 화면에 즉시 반영
강조 역량
실사용 문제 정의 · 실시간 DB 설계 · 사용자 역할 분리 · 보안/편의 균형 · 학교 현장형 서비스 구현
