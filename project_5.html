왜 만들었나

문제의식
교통사고 = 복합 위험 요인
운전자 부주의뿐 아니라 기상 조건·시간대·지역별 사고 빈도·사고 심각도가 복합적으로 작용한다.
기존 방식의 한계
"비 오면 위험하다"는 직관
정성적 판단을 데이터 기반 수치로 검증 가능한 형태로 바꿀 필요가 있다.
자율주행 연결
도로 환경 위험 판단 구조
자율주행 안전 판단 알고리즘의 핵심 구조인 "현재 환경 위험도 계산"과 연결된다.

기상 정보 수집

한계 및 개선 방향

솔직한 한계 분석

현재 한계
①교통사고 CSV는 집계 통계라 개별 사고 시각과 기상 데이터를 직접 매칭 불가
②초단기실황 API는 시정거리 미제공 → 실시간 대시보드에서 안개 위험도 직접 반영 한계
③사고건수가 많은 대도시가 높게 나타나는 구조 → 인구·교통량 보정 변수 필요
④현재 가중치는 탐구 목적 기준, 교통공학적 검증을 거친 공식 모델 아님

가중치 기반 위험도 산출
4개 요소 × 가중치로 0~100점 산출. 안개 조건 시 위험도가 단순 강수보다 높다는 것을 데이터로 검증.
웹 시각화 · API 연동
Flask 풀스택 구현
Flask + Chart.js + 기상청 API + 캐시 + 마스킹까지. 인증 오류·호출 제한·보안을 직접 해결.
