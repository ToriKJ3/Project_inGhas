문제의식
기상 조건만으로는 부족하다
악천후·야간 주행의 실제 위험도는 지역별 사고 빈도, 심각도, 기상 조건을 함께 봐야 한다.
컴공 역량 적용
데이터 수집부터 시각화까지
공공데이터 수집 → API 연동 → 전처리 → 가중치 알고리즘 → CSV 출력 → 그래프
자율주행 연결
진로와 연결되는 관점
자율주행차가 도로 위험도를 판단하려면 날씨·시간·지역 위험도를 해석하는 소프트웨어가 반드시 필요하다.

분석 결론

세 가지 핵심 발견

결론 01

안개·시정거리 저하가 단순 강수보다 평균 최종 위험도를 더 높게 만든다. (안개 61.5 vs 비 41.7)

결론 02

같은 기상 조건에서도 수원시처럼 사고 빈도·심각도가 높은 지역은 최종 위험도가 더 높게 산출된다.

결론 03

CSV 집계 통계 + API 시간자료를 결합해 "지역 기반 위험"과 "시간 기반 위험"을 분리하여 계산 가능하다.

한계 및 개선 방향
① 사고 데이터가 시군구별 집계 통계라 개별 사고 시각과 직접 매칭 불가
② ASOS 관측소가 모든 시군구에 없어 인접 대표 지점 사용
③ 교통량·도로 형태·실시간 초단기실황 API 추가 시 모델 정밀화 가능
